CRITICAL ISSUES (Must Fix Before Production):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. [HIGH] XSS Vulnerability in Regex Tester
   - dangerouslySetInnerHTML with incomplete escaping
   - Single quotes not escaped → HTML attribute injection
   - FIX: Update escapeHtml() to escape single quotes and slashes
   
2. [HIGH] ReDoS Attack Vector in Regex Tester
   - No execution timeout on user-provided regex patterns
   - Catastrophic backtracking can freeze browser
   - FIX: Add 1000ms timeout protection
   
3. [HIGH] Privacy Violation in Glass Generator
   - Loads images from images.unsplash.com
   - Leaks user IP address to external CDN
   - FIX: Replace with local background images
   
4. [MEDIUM] Script Injection in SVG to JSX
   - Doesn't strip <script> tags from SVG input
   - User copies malicious JSX → XSS in their app
   - FIX: Sanitize SVG input, strip scripts and event handlers
